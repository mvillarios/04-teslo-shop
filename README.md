# Teslo Shop API

1. Instalar dependencias

```
yarn install
```

2. Clonar el archivo de variables de entorno

```
cp .env.template .env
```

4. Cambiar Variables de entorno en el archivo .env

5. Levantar la base de datos con docker-compose

```
docker-compose up -d
```

6. Levantar la aplicación

```
yarn start:dev
```
